Escape the Boba Shop

Roblox Obby

Over summer I decided to create my own game on roblox. I had never created a game before, so I wanted it to be something simple, that still required some programming, so I decided on an obby.

My obby had over 20 levels each themed around a boba shop, it also features checkpoints and gamepasses which were programmed by me in the language roblox studio uses (Lua)
